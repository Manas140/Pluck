# Pluck

A X11 Color Picker

<p align="center">
  <img src="preview.png">
</p>

Dependencies:

```
  xinput
  xdotool
  ImageMagick
  notify-send (optional)
  xclip (optional)
```

Install:

```
  git clone https://github.com/Manas140/pluck.git && cd pluck
  ./install.sh i
```
Usage:

```
  pluck [rgb|h]
    rgb: use rgb value instead of hex
      h: help
```

Example:

```
  pluck
```
